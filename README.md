# ICCC-320-Data-Networked-Interaction-Fall-2026
Data & Networked Interaction is a course that introduces students to real-time data, network communication, and event-driven interaction as core materials for contemporary design and art practice. At the undergraduate level, the course emphasizes foundational literacy: students learn to work with API and JSON-based data workflows, use event-driven network protocols, build simple networked interactive systems, and design interfaces that translate live data into visual and interactive media. At the graduate level, the focus shifts toward systems authorship and critical practice. Graduate students design robust data schemas and pipelines, implement advanced event-driven network architectures, integrate live data from heterogeneous sources, and build multi-device, networked interaction systems.
